# Sravya Speech & Hearing Clinic Website

Business website for Sravya Speech & Hearing Clinic, Thrissur, Kerala.
Built with React + Vite + Tailwind CSS + shadcn/ui. Deployed on Vercel.

## Getting Started
```bash
npm install
npm run dev        # http://localhost:3000
npm run build      # production build → dist/apps/web
```

## Stack
- React 18, React Router v7, Vite 7
- Tailwind CSS 3, shadcn/ui, Framer Motion
- React Hook Form + Zod


## Deployment
Auto-deploys to Vercel on every push to main.
Build root: apps/web | Output: ../../dist/apps/web
